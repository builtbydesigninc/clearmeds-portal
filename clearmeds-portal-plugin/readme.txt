Backend plugin for ClearMeds affiliate portal - manages affiliates, referrals, commissions, and payouts.

== Description ==

ClearMeds Portal is a comprehensive WordPress plugin that provides backend functionality for the ClearMeds affiliate portal Next.js application. It integrates with WooCommerce to track orders, calculate commissions, and manage affiliate networks.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/clearmeds-portal-plugin` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure settings in Settings > ClearMeds
4. Ensure WooCommerce is installed and activated for order tracking

== Frequently Asked Questions ==

= Does this require WooCommerce? =

Yes, WooCommerce is required for order tracking and commission calculation.

= How do I configure commission rates? =

Go to Settings > ClearMeds in the WordPress admin area.
